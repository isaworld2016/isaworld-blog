## Atomic Design Pattern
* atom(원자): 위치 속성을 가지지 않음.
* molecule(분자): 원자를 엮어 조금 복잡한 덩어리. 하위 레벨의 위치값 조정.
* organism(유기체): 분자가 되지 않은 원자가 엮이기도. 하위 레벨의 위치값 조정.

## Code Convention
