
## 2. BEM 뿌리기
이걸로는 부족하다!
[BEM](https://velog.io/@nemo/bem)을 뿌리자.

> BEM이란 Block, Element, Modifier의 약자
- class명을 종속적으로 기술하는 게 아니라 블럭으로 구성.
- 여기서 블럭은 모듈/컴포넌트처럼 '독립적으로 존재하는 덩어리' 라고 생각하면 됨.
- "블럭" __ "엘리먼트" -- "유니크한 속성"


## 3. After
어떻게 바뀌었을까!

```css
@mixin icon__a {
  display: -webkit-box;
  display: flex;
  margin: 0 0 0 20px;
  font-family: "Inter",sans-serif;
  font-size: 15px;
  font-weight: bold;
  cursor:pointer;
}
@mixin icon--rotate {
  transform:rotate(180deg);
}

.pagination{
  display: flex;
  padding: 20px 0 60px 0;
  position:relative;
  width: 100%;
  justify-content: center;
  -webkit-box-pack: center;

  &__wrap{
    display: flex;
    align-items: center;
    -webkit-box-pack: center;
    justify-content: center;
    height: 43px;
    border-width: 1px;
    border-style: solid;
    border-color: #000;
    border-radius: 6px;
    padding: 0 20px 0 20px;

    .icon__previous{
      display: flex;
      a{
        @include icon__a;
      }
      &--rotate-180{
        @include icon--rotate;
      }
    }
    .icon__next{
      display: flex;
      a{
        @include icon__a;
        margin: 0 20px 0 0;
      }
      &--rotate-180{
        @include icon--rotate;
      }
    }

    .nav {
      display: -webkit-box;
      display: flex;
      -webkit-box-align: center;
      align-items: center;
      -webkit-box-pack: center;
      justify-content: center;

      &__links{
        margin: 2px 0 0 0;
        display: -ms-flexbox;
        display: flex;
        &:hover{
          text-decoration: none;
        }

        .nav__numbers {
          display: inline-block;
          margin: 0 8px;
          font-family: "Inter",sans-serif;
          font-size: 15px;
          font-weight: bold;
          padding: 0 10px;
          color: #000;
          cursor:pointer;
          &:hover{
            text-decoration: underline;
            text-decoration-thickness: 3px;
          }
          &--cursor-unset:hover{
            cursor: unset;
            text-decoration: none;
          }
          &--current{
            color:red;
            cursor:initial;
            text-decoration: none;
          }
        }
      }
    }
  }
}
```

## 4. 느낀점
쌩 css를 써 보려고 했으나,

각 모듈마다 세세히 쪼개지면서 클래스명을 붙였더니
머리속에 명쾌하게 정리되는 기분이다!
